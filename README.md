## Overview
PostureSense is a full-stack web application that analyzes user posture in real-time using webcam input and provides feedback.

## Tech Stack
Frontend: React, TypeScript, TailwindCSS  
Backend: FastAPI, Python  
Computer Vision: MediaPipe  
Database: PostgreSQL  

## Features
- Real-time webcam posture analysis
- Video upload posture analysis
- REST API backend
- Modern responsive UI
- Posture scoring and feedback system

## Architecture
React frontend communicates with FastAPI backend via REST APIs. Backend processes posture using MediaPipe and returns analysis data.

- Real-time frontend-backend integration
- Handling webcam streams
- Designing scalable REST APIs
